##  О проекте

**Honduras Orbital Program (HOP)** — это информационная веб-платформа для поддержки национальной космической инициативы Гондураса. Проект предполагает создание наноспутника формата CubeSat для мониторинга погодных изменений и наблюдения Земли с орбиты.

Сайт служит инструментом для:
-  Информирования общественности о космической программе
-  Привлечения спонсоров и партнёров
-  Регистрации участников и управления личными кабинетами
-  Подписки на новости проекта
-  Сбор пожертвований на реализацию проекта

##  Функциональность

### Информационные страницы
- **Главная** — обзор проекта с интерактивной каруселью истории космической гонки
- **О проекте** — детальная информация о миссии HOP
- **Благотворительность** — возможности поддержки проекта
- **Команда** — информация об участниках

###  Система авторизации
- Регистрация и вход пользователей
- Личный кабинет с редактированием профиля
- Загрузка и предпросмотр аватара
- Защита от CSRF-атак
- Хеширование паролей (PBKDF2)

### Подписка на рассылку
- Форма подписки на главной странице
- Проверка уникальности email
- Интеграция с личным кабинетом

###  Поддержка проекта
- Информация о вариантах пожертвований
- Интеграция с внешней системой донатов

###  Административная панель
- Управление пользователями
- Просмотр подписчиков
- Модерация контента
